Dispositivos equipados com processadores Qualcomm Snapdragon geralmente oferecem compatibilidade superior devido aos drivers gráficos maduros Turnip.

<Section icon={Map} title="Guia de Configuração Passo a Passo do Winlator">

Configurar o ambiente de emulação requer a preparação dos arquivos do jogo em um PC e a transferência deles para o seu dispositivo Android. Siga estas etapas para inicializar o container e configurar o ambiente de execução.

<Steps>
  <Step num="1" title="Prepare os Arquivos do Jogo">
    Instale o jogo no seu PC. Localize o diretório de instalação e copie toda a pasta do jogo para o armazenamento interno do seu dispositivo Android, de preferência no diretório **Download** para facilitar o acesso.
  </Step>
  <Step num="2" title="Crie um Container no Winlator">
    Abra o emulador e toque no ícone "+" para criar um novo container. Defina o nome do container, selecione a resolução da tela (comece com **800x600** para um desempenho ideal) e escolha o driver gráfico apropriado.
  </Step>
  <Step num="3" title="Configure as Configurações de DXVK e Wine">
    Nas configurações do container, navegue até a guia DXVK. Escolha **DXVK 1.10.3** ou mais recente. Defina as configurações do registro do Wine para emular um ambiente Windows de 64 bits e atribua todos os núcleos de CPU disponíveis ao container.
  </Step>
  <Step num="4" title="Configure os Controles de Entrada">
    Navegue até o menu de Controles de Entrada (Input Controls). Crie um novo perfil e mapeie as teclas virtuais para corresponder ao layout padrão do jogo (WASD para movimentação, clique do mouse para atirar e barra de espaço para pular). Salve o perfil.
  </Step>
  <Step num="5" title="Iniciar e Otimizar">
    Inicie o container, navegue até a pasta do jogo através do gerenciador de arquivos integrado e execute o executável principal. Assim que o jogo iniciar, abra imediatamente o menu de opções para ajustar as configurações internas.
  </Step>
</Steps>

<Callout type="tip" title="Seleção de Driver">
Para dispositivos Snapdragon, sempre selecione o driver gráfico **Turnip (Adreno)** nas configurações do container. Este driver fornece a camada de tradução Vulkan mais estável e evita artefatos gráficos.
</Callout>

</Section>

<Section icon={Swords} title="Otimização Gráfica e Configurações">

Para manter uma taxa de quadros estável durante sequências de combate intensas, você deve equilibrar as configurações do container e as opções internas do jogo. Configurações abaixo do ideal podem levar a vazamentos de memória, travamentos ou engasgos graves.

A tabela abaixo destaca a configuração recomendada para o container dentro da interface do emulador.

| Categoria de Configuração | Opção Recomendada | Objetivo |
|---|---|---|
| **Driver Gráfico** | Turnip (Adreno) | Maximiza a compatibilidade de renderização |
| **Versão do DXVK** | DXVK 1.10.3 | Estabiliza a tradução do DirectX 9/11 |
| **Driver de Áudio** | ALSA / PulseAudio | Minimiza a latência e estalos de áudio |
| **Afinidade de CPU** | Selecionar todos os núcleos exceto o Núcleo 0 | Evita travamentos na interface do sistema |
| **Predefinição do Box64** | Compatibilidade / Intermediário | Equilibra velocidade de execução e estabilidade |

Uma vez dentro do jogo, ajuste as configurações de vídeo para reduzir a carga sobre a GPU emulada. Use as configurações recomendadas abaixo para obter o melhor equilíbrio entre clareza visual e desempenho.

| Configuração no Jogo | Valor Recomendado | Impacto no Desempenho |
|---|---|---|
| **Resolução** | 800x600 ou 1024x768 | **Alto** - Reduz drasticamente a taxa de preenchimento de pixels |
| **Qualidade de Textura** | Médio | **Médio** - Economiza alocação de VRAM |
| **Qualidade das Sombras** | Baixo / Desativado | **Alto** - Minimiza cálculos complexos de iluminação |
| **Anti-Aliasing** | Desativado | **Alto** - Evita a sobrecarga de suavização de bordas |
| **Profundidade de Campo** | Desativado | **Baixo** - Melhora a visibilidade em telas pequenas |

<Callout type="success" title="Ganho de Desempenho">
Desativar as sombras e reduzir a resolução para 800x600 pode melhorar a estabilidade geral dos quadros em até **35%** em chipsets intermediários.
</Callout>

</Section>
