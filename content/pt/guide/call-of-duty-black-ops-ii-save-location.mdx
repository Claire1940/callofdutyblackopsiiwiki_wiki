Gerenciar seus dados de progressão é essencial para jogadores que desejam experimentar diferentes desfechos de missões ou restaurar o progresso perdido. Na versão para PC do jogo, a estrutura de diretórios é relativamente simples.

<Section icon={FolderDown} title="Instalando Arquivos de Salvamento Personalizados">

**Passos para Instalação:**

<Steps>
  <Step num="1" title="Fazer Backup dos Dados Atuais">

  </Step>
  <Step num="2" title="Extrair Arquivos Baixados">
    Extraia o conteúdo do arquivo zip de salvamento baixado em uma pasta temporária.
  </Step>
  <Step num="3" title="Substituir o Diretório">
    Mova ou copie a nova pasta "players" para o diretório raiz do seu jogo, confirmando a solicitação de substituição.
  </Step>
  <Step num="4" title="Verificar Integridade">
    Inicie o jogo para garantir que o seu novo status de salvamento seja refletido corretamente no menu da campanha.
  </Step>
</Steps>
